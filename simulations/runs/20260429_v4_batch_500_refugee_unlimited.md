# v4 批量 500 局 — 难民修复后

## Bug 修复

`refugee_used` flag 错误地把难民效果限制为"每局一次"，但 JSON 没说"once per game"。
修复：移除 flag，难民效果可通过傀儡死亡链重复触发，仅受"善良死人池"约束。

## 验证

100 局测试：
- 难民触发次数分布: 0, 1, 2, 3
- **28% 局触发 ≥2 次**（之前 0%）
- 最高 3 次/局

## 总体

- 善良: **25.0%** / 邪恶: 75.0% / 超时: 0%
- 平均天数: 6.4

## 解读

整体差异在统计噪声范围内（±2.5 pp 95% CI on 500 games），但分恶魔类型有信号：
- **征服者 −7.4 pp**: 难民可重复触发 → 末日时邪恶死票更多，反胜门槛更难达到
- **暗箭手 +4.9 pp**: reasoner 看到多次"难民触发"识别邪恶模式更准
- **千面人翻倍** (3.3 → 6.5): 多 demon 时难民触发需要善良死人，但千面人前期慢节奏让重复触发机会少
