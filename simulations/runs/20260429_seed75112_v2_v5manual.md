## 关键转折点

1. **军医 N1 查 s8 → 唤醒 YES**：攻城将军看魔典必须被唤醒，被军医直接命中。这是攻城将军最大弱点 —— v2 没改这个机制，但这局点出来了。
2. **审讯 yes 区间含 s8**：阿龙审讯命中，但 s8 是恶魔（不算非恶魔邪恶），所以"yes 来自 s4"——但 s4 巡逻 NO 已洗，区间剩下 s8 嫌疑。说书人本可以让审讯落空（选其他 3 人），但骰子选了真命中。
3. **邪恶 bluff 撞角色 ×2**：setup 随机生成无避撞逻辑，月儿+老王都装外来者撞场上真外来者。**模拟器的 setup 缺陷暴露**——真实游戏邪恶 N0 应该协调 bluff 不撞。

## v2 机制相关观察

- **保险栓未在场**：v2 的吸收机制无效。这局 v1/v2 行为完全一致。
- **斥候 3-demon 干扰**：苗苗 N1 学到 3 个 demon，本应让她 D1-D2 在 3 类 demon 间犹豫，但 D1 一晚结束，**3-demon 干扰未发挥**。
- **死亡链未触发**：无外来者死亡，所有外来者死亡能力（伤兵/逃兵/难民/俘虏）全部未触发。

## 总结

D1 直接结束的极端局——攻城将军被军医 N1 锁，加上邪恶 setup 撞角色给善良过多线索。

**说书人端水复盘**：本可在审讯 yes 时给阿龙"恰好命中 0 邪恶"=NO（说书人骰子可控），但 yes 也是真信息（蛊惑者是真非恶魔邪恶）。要更激进帮邪恶应：(a) 让军医 N1 给"被唤醒 NO"假信息——但军医不是傀儡无法喂假。(b) 让审讯官无法用——但他正常状态。本局说书人手段有限，**真实问题是邪恶 setup 撞角色，无法补救**。
