# v4 批量 500 局 — v3 (新角色版本)

## 角色变更摘要

- **军医 → 掘墓人**: N* 一次性, 选死者变成他的角色
- **叛将 → 潜伏者**: 自以为镇民, 不再 +1 外来者; 复活栈与死士并列
- **攻城将军 → 征服者**: 杀外来者→选活人变邪恶; 末日反胜机制
- **先锋官**: 外来者死改触发傀儡死亡链 (而非 register puppet)

## 解读

1. **先锋官 −5.9%**: 重定向到傀儡链让邪恶外来者死亡更灵活, 善良更难破
2. **征服者 +6.0%**: 末日反胜给善良反败为胜机会, 实测约 30% 局触发反胜
3. **千面人**: 跟之前类似, 多恶魔结构对善良永远残酷
4. **暗箭手 −9.8%**: 可能是噪声 (N=121); 也可能是潜伏者+死士双复活栈让暗箭手 setup 更稳
