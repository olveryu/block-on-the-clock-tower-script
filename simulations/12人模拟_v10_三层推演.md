### 胜负

|---|---|---|---|

### 关键节点

1. **N0 邪恶 bluff 默契分配**（不需要私聊协调，眼神搞定）—— 阿强=瞭望兵、月儿=巡逻兵、阿龙=密探
2. **N1 苗苗审讯官 1/4/11 yes** + **雪儿纹章 yes**——双链直接锁定邪恶非恶魔 = 月儿 或 阿强（其中阿信外来者排除）
3. **D1 私聊阶段** —— 苗苗+雪儿+大刘形成信任小圈，互相验证信息
4. **邪恶 D1 没主动找队友私聊** —— 月儿月亮想给阿强信息但用眼神，符合社区共识
5. **公开推理**：阿强装瞭望兵+先锋官 setup 锁恶魔类型 → **阿强=先锋官嫌疑高**
6. **月儿主动求死**——想用内应傀儡触发链给邪恶续命，但雪儿识破"主动求死=想触发链"
7. **D1 处决阿强** = 直接恶魔死 = **善良 D1 速胜**

### 反思——这局为什么这么短

**配置上对善良超有利**：
- 苗苗审讯官 N1 选了**月儿+阿强**（两个邪恶都在 1/4/11）
- 雪儿纹章 D1 立刻验证（恰好一外来者一致）
- 大刘斥候 N1 锁定恶魔类型
- 阿强装瞭望兵（先锋官 setup 验证）→ 顺锁恶魔

**邪恶失败原因**：
- 苗苗审讯刚好选中两个邪恶（**纯运气**）
- 阿强装瞭望兵报"叛将不在场"→ 反而暴露 setup = 先锋官 = 自己
- 月儿"主动求死想触发链"被雪儿识破

**本局核心教训**：
- **审讯官 N1 选 3 个里命中 2 个邪恶 = 邪恶 D1 难逃**
- **邪恶装 bluff 时要避免暴露 setup 来源**——阿强如果装"巡逻兵"或"密探"反而更安全（不需要给 setup 信息）

### 三层推演的效果

引入**私聊优先**让模拟更真实：
- 私聊场景占 D1 大部分篇幅
