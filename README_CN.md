## 剧本简介

**围城之夜**是一个以**傀儡**为核心机制的高级剧本。在这个剧本中，外来者死亡会触发强大的连锁效果——但代价是由恶魔来选择如何执行。善良阵营必须谨慎决定谁该活、谁该死，而邪恶阵营则想方设法触发外来者的死亡效果来制造混乱。

### 核心特色

🏰 **傀儡生态**
外来者死亡时触发连锁效果：额外击杀（逃兵）、阵营转换（难民）、创造傀儡（伤兵）、处决绑定（俘虏）。傀儡死亡还能再触发外来者效果——连锁可以滚雪球。

🛡️ **双重保护（吸收型）**
盾卫吸收外来者夜死的连锁——自己变傀儡，原效果不触发。女伯爵同理，针对处决。一次有效，且保险栓本人不知情，变傀儡后仍以为自己有能力。

🎭 **四种截然不同的恶魔**
- **征服者** — 杀外来者策反活人，终局指认翻盘型
- **暗箭手** — 首夜制造傀儡，可交换角色隐藏身份
- **先锋官** — 外来者死亡时改为触发傀儡的死亡效果，连锁爆发型
- **千面人** — 所有爪牙都变成千面人，多恶魔协作击杀

🕵️ **丰富的信息角色**
审讯官（查邪恶）、纹章官/巡逻兵（查外来者）、掘墓人（继承死者能力）、牧师（查角色变动）、书记官（外来者+爪牙座位号总和）、斥候（查恶魔类型）——善良有很多工具，但蛊惑者随时可能让你的信息变假。

## 玩法指南

### 善良阵营

**你的目标：处决恶魔。**

1. **保护信息**：不要第一天就公开你的角色。先通过私聊建立信任圈，逐步分享信息。高价值角色（斥候、女伯爵、盾卫）尤其要隐藏——公开就会被夜杀。

2. **警惕蛊惑者**：蛊惑者每晚让一名邻座变成傀儡（=中毒），你收到的信息可能是假的。收到异常信息时，先考虑"我被蛊惑了吗？"而不是盲目信任。

3. **利用外来者连锁推理**：如果首次外来者死亡时没有连锁→盾卫/女伯爵被触发并已变傀儡。但**仅首次有效**，之后连锁正常发动。变傀儡的盾卫本人不知情，可能继续公开声称"我是盾卫"——这反而是邪恶的烟雾弹。

4. **定位蛊惑者**：蛊惑者只能选邻座。如果你的信息出错，看看你的邻座——蛊惑者可能就在附近。

5. **小心傀儡**：傀儡以为自己是善良角色。如果有人的信息持续矛盾，他可能是傀儡而不自知。

### 邪恶阵营

**你的目标：存活到最后（仅剩2名存活玩家时邪恶获胜）。**

1. **主动出击**：私聊善良玩家，交换假信息，套出他们的角色。得知关键角色位置后告诉恶魔精准击杀。

2. **制造"蛊惑者恐慌"**：善良最怕信息不可靠。推动"蛊惑者在那边"的叙事，让善良自相残杀。

3. **互相举报**：邪恶A提名邪恶B→B被处决→善良信任A→A继续潜伏。这是经典且有效的战术。

4. **利用连锁**：故意把外来者死亡效果用到最大价值——制造更多傀儡、转化善良玩家、让邪恶玩家获得新角色。

5. **假冒关键角色**：假女伯爵（让善良以为处决安全）、假牧师（编造查角色信息）、假斥候（误导恶魔类型判断）。

### 关键提示

1. **夜晚结算严格按顺序**：按 firstNight / otherNight 数字从小到大。蛊惑者(2)→恶魔(6-9)→外来者死亡效果(10-13)→镇民信息(14-21)。

2. **蛊惑者的傀儡效果**：被蛊惑者选中的玩家在"当晚和明天白天"变成傀儡。这意味着当晚的信息是假的，明天白天的能力也失效。

3. **连锁与吸收**：当外来者夜死且盾卫存活、未中毒、未醉酒时——盾卫立刻变傀儡（魔典内换标记，不告知本人），外来者死亡效果不触发。女伯爵同理，针对处决。一次后保险栓失能，但本人继续以为自己有能力（按傀儡守则继续欺骗：照常夜唤、提供"无外来者死"等假反馈）。

4. **死士的特殊性**：死士被视为外来者，但他不是外来者——纹章官/巡逻兵会检测到他，但他死亡时触发的是傀儡效果而非外来者效果。

5. **千面人局的信息处理**：千面人被视为外来者和爪牙。审讯官查到千面人=阳性（非恶魔邪恶）。纹章官查到千面人=阳性（外来者）。军需官选千面人=醉酒（爪牙）。

6. **最后一个恶魔被处决→游戏立刻结束**，连锁效果来不及生效（除非死士在场继承恶魔）。

## 设计理念

围城之夜的核心张力在于：**外来者既是善良的负担，也是邪恶的资源。**

善良想保护外来者不死（避免连锁），但外来者占了善良的名额却没有主动能力。邪恶想杀外来者触发连锁（制造傀儡、转化阵营）。盾卫/女伯爵能各吸收一次连锁，但代价是自己变成新的傀儡弹药——之后被杀仍可触发外来者死亡效果。

这创造了一个独特的博弈：**处决谁、夜杀谁、保护谁，每个决定都牵一发而动全身。**
